# SendaFlip - Riskantne mündiviskemäng
 
- Riskantne mündiviske mäng, kui võidad siis on hästi aga kui kaotad... noh ära kaota tolbik

# Projekti kirjeldus c:

SendaFlip on veebirakendus, mis demonstreerib 4 DevOps tööriista integreerimist:
- Git - Versioonihaldus
- GitHub Actions - CI/CD pipeline  
- Docker - Containeriseerimine
- Ansible - juurutamine

# Kuidas mängida?

- Logi virtuaalmasinasse sisse
- Mine ssh'ga windowsisse
- Seejärel mine sendaflip kasuta - cd sendaflip
- Käivita virtuaalkeskkond source/ven/bin/activate
- Käivita rakendus python3 app.py
- Mine veebilt oma masina ip aadressile ja lisa lõppu enda määratud port. Minul on 10.0.82.10:5000
- Ja mängi mängu!


# tööriistad

# 1. Git (kohustuslik)
Miks: Versioonihaldus, koodi jälgimine, koostöö

### 2. GitHub Actions (kohustuslik)  
Miks: CI/CD pipeline - automatiseeritud testid, build, deploy

### 3. Docker (enda valik)
Miks: Containeriseerimine tagab ühtse keskkonna kõikjal
Kuidas integreerib: Dockerfile, docker-compose.yml, GitHub Actions build

### 4. Ansible (enda valik)
Miks: automatiseeritud juurutamine
Kuidas integreerib: Playbook'id serveri seadistamiseks ja rakenduse deploy'ksVeebirakendus mündiviskemänguks. Vali tulemus, viska münt ja testi oma õnne!
